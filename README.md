# 911-calls-Kaggle-
Emergency (911) Calls: Fire, Traffic, EMS (Emergency Medical Services) for Montgomery County, PA

Data from Kaggle.  
Where:  
* lat : str, Latitude 
* lng: str, Longitude
* desc: str, Description of the Emergency Call 
* zip: str, Zipcode 
* title: str, Title 
* timeStamp: str, YYYY-MM-DD HH:MM:SS 
* twp: str, Township 
* addr: str, Address 
* e: str, Dummy variable (always 1)
